> \*\*Look up. Fly higher.\*\* Skyz Client is a free, open-source, and highly customizable Minecraft: Java Edition client built on Fabric.



\---



\## ✨ Key Features



\### 🚀 Performance



\* Sodium (v0.8.2)

\* Lithium (v0.14.3)

\* FerriteCore (v7.0.0)

\* Iris (v1.10.4)



\### 🎨 UI



\* Glassmorphism design

\* HUD editor

\* Themes (Midnight, Aurora, Sakura)

\* Animated menus



\### 🛠️ Modules



\* Skin editor

\* Cosmetics

\* Friends/chat system

\* PvP tools (CPS, keystrokes)
